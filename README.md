### Bank Tech Test



Week 10 Makers Academy - tech test to build a banking application.

# Technology used

- Ruby
- RSpec
- Rubocop
- Simple Coverage

# Setup

``` 
$ git clone https://github.com/kristycer/bank-tech-test.git
$ cd bank-tech-test
$ bundle
$ irb
> require ‘._lib_account.rb’  

```
